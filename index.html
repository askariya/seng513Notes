HTML/CSS Notes: Lecture

Basic Tags

script tag

The script tag is used to embed JavaScript code into your website

noscript tag

The noscript tag is used to define a section of HTML that will be inserted if a script is not supported or disabled.
For example: to display an error.

meta tag

Contains other metadata. Can also use to refresh the page (but avoid doing this).
For example: keywords, author, etc.

Headings

Six different headings: h1 to h6.

Lists

Two different types:

  1. Ordered: ol (what this is)
  2. Unordered: ul (basically point form)
Can use CSS to change the bullet type (circle, disc, square, etc.)

blockquote and pre

  • blockquote: text looks pretty lame
  • pre: basically a paragraph where white space is acknowledged

Tables

Can stylize with CSS: table { font: etc.}

div

  • Div: div is basically just used for styling with CSS sheets
  • Span: span is used to specify ID or Class when styling

Classes and IDs

  • Classes: Used to group similar components that you want to share JS or CSS.
    An element can belong to multiple classes.
  • IDs: Similar to Classes but IDs are unique and used to reference just 1 specific element

Relative Measurements

em and rem

Use "em" instead of px.
em = size relative to font of the element. rem = size relative to the font of the root element (not commonly used)

Percent (%)

Can specify width and height of elements as percentages.
For example: width: 50% would make the element half the size of the page even when the page is resized.
