HTML/CSS Notes: Tutorial

Working with Text

br tag

Can use this to break line (done above). However the correct way to do it would be to use paragraphs (p). Enclose what should be in a paragraph in p brackets. Note that you can use the style identifier to change the width of the paragraph in-line.

However, it is easier just to define align and width for paragraphs in the CSS.

strong

Like bold but has more meaning semantically (strong importance). Browsers can recognize this.

em

Like italics but has more meaning semantically. Browsers can recognize this.
